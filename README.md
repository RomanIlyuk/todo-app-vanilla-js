## Description

A simple educational To-Do List app that allows you to:

- Add new tasks;
- Edit tasks;
- Delete individual tasks;
- Clear all tasks at once;
- Store tasks in **localStorage**, so they persist after page reloads.

This project was created for learning purposes to practice DOM manipulation, event handling, and localStorage in JavaScript.

---

## Features

1. **Add Tasks**

   - Using the **Add** button;
   - By pressing **Enter** in the input field.

2. **Edit Tasks**

   - Tasks can be edited directly in the input field next to the **Delete** button.

3. **Delete Tasks**

   - Individual tasks can be removed with the **Delete** button;
   - All tasks can be cleared with the **Clear All** button.

4. **Persistence**
   - Tasks are stored in the browser's **localStorage**;
   - All tasks are restored automatically when the page is reloaded.
